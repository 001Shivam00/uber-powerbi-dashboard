# 🚕 Uber Ride Analysis Dashboard - Power BI

## 🔹 Project Overview
This project analyzes Uber ride data to uncover revenue patterns, ride trends, and customer behavior using a clean black & white dashboard theme.

## 🔹 Tools Used
- Power BI
- DAX
- Data Cleaning
- CSV Dataset

## 🔹 Key Metrics
- Total Rides
- Total Revenue
- Average Fare
- Peak Hour Analysis

## 🔹 Features
- Interactive Filters
- Minimal Black & White Theme
- Business Insights Visualization
